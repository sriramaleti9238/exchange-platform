# Exchange platform

Currency exchange platform in market exchange model where user may submit their currency exchange
offers.

> [!IMPORTANT]
> Project is in a pre-alpha state, and only suitable for use by developers
>
>

## Technologies

* PostgreSQL
* Kafka
* SpringBoot
* Liquibase
